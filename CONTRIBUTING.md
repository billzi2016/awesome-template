# Contributing

Thank you for helping improve this project.

## What to Submit

Good contributions include:

- resources directly relevant to the project's stated scope;
- official code repositories;
- maintained libraries or toolkits;
- tutorials, lecture notes, or technical guides;
- benchmarks, datasets, or evaluation protocols with clear relevance;
- applications that clearly belong to the topic covered by this repository.

Please do not submit:

- resources unrelated to the project scope;
- generic background material with only weak relevance;
- promotional pages without technical detail;
- copied PDFs, mirrored datasets, or unofficial model-weight uploads;
- links that cannot be verified.

## Entry Format

Paper entry:

```md
- **Paper Title** - Author et al., Venue Year. One-sentence objective description of the method, task, or contribution. [[Paper](link)] [[Code](link)]
```

Code entry:

```md
- [Repository Name](link) - One-sentence objective description of scope, implementation, and use case.
```

## Pull Request Checklist

- [ ] The resource is directly relevant to the project scope.
- [ ] The link is real and accessible.
- [ ] The title, authors, year, and supporting links are correct.
- [ ] The description is objective and concise.
- [ ] The item is not a duplicate.
- [ ] The entry follows the existing format.
- [ ] The English and Chinese README files are kept in sync when needed.
- [ ] The resource does not introduce copyright, license, privacy, or platform-policy risk.

## Link and Copyright Requirements

Use original sources whenever possible:

- arXiv, DOI, publisher, or official paper page;
- official author or organization repository;
- official dataset page;
- official documentation;
- official benchmark or project page.

Do not copy copyrighted PDFs, books, course materials, datasets, model weights, or images into this repository.

## Language Sync

This project uses:

- `README.md` for English;
- `README.zh-CN.md` for Simplified Chinese.

When adding or removing resources, keep both files aligned unless the change only affects one language-specific explanation.
