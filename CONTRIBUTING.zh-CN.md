# 贡献指南

感谢你帮助改进本项目。

## 可以提交什么

适合提交的资源包括：

- 与本项目主题范围直接相关的资源；
- 官方代码仓库；
- 持续维护的库或工具；
- 教程、课程笔记或技术指南；
- 有明确相关性的 benchmark、数据集或评估协议；
- 明确属于本仓库主题范围的应用。

请不要提交：

- 与项目范围无关的资源；
- 只有弱相关性的泛背景材料；
- 没有技术细节的宣传页面；
- 复制的 PDF、镜像数据集或非官方模型权重；
- 无法核验的链接。

## 条目格式

论文条目：

```md
- **论文标题** - 作者，会议/期刊 年份。一句话客观说明方法、任务或贡献。[[论文](link)] [[代码](link)]
```

代码条目：

```md
- [仓库名称](link) - 一句话客观说明范围、实现和使用场景。
```

## PR 检查清单

- [ ] 资源与项目主题范围直接相关。
- [ ] 链接真实且可以访问。
- [ ] 标题、作者、年份和相关链接正确。
- [ ] 描述客观、简洁。
- [ ] 不是重复条目。
- [ ] 条目格式符合现有规范。
- [ ] 如有必要，英文和中文 README 已同步更新。
- [ ] 资源不会引入版权、许可证、隐私或平台规则风险。

## 链接与版权要求

优先使用原始来源：

- arXiv、DOI、出版社页面或官方论文页；
- 作者或机构官方代码仓库；
- 数据集官方页面；
- 官方文档；
- benchmark 或项目官方页面。

不要把受版权保护的 PDF、书籍、课程材料、数据集、模型权重或图片复制到本仓库。

## 双语同步

本项目使用：

- `README.md` 作为英文版本；
- `README.zh-CN.md` 作为简体中文版本。

新增或删除资源时，请保持两个文件内容同步，除非该修改只影响某个语言版本的说明文字。
