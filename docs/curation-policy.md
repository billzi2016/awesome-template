# Curation Policy

This document defines how resources are included in this project.

## Inclusion Criteria

A resource may be included when it satisfies most of the following:

- directly relevant to the repository's stated scope;
- has a stable and accessible source;
- provides a paper, codebase, documentation, benchmark, dataset, or technical explanation;
- is useful for research, engineering, evaluation, or learning;
- has an objective description that can be verified from the source;
- does not introduce copyright, license, privacy, or platform-policy risk.

## Priority Resources

Higher priority is given to:

- foundational papers;
- official implementations;
- maintained libraries;
- reproducible experiments;
- widely used tutorials or notes;
- benchmark and evaluation resources;
- applications that clearly belong to the repository topic.

## Rejection Criteria

A resource should not be included when it is:

- only weakly related to the repository scope;
- inaccessible or unverifiable;
- a duplicate of an existing item;
- mainly promotional;
- missing technical detail;
- a copied PDF, mirrored dataset, or unofficial model-weight upload;
- likely to violate copyright, licenses, privacy, or platform rules.

## Link Policy

Prefer original sources in this order:

1. arXiv, DOI, publisher, or official paper page
2. official author or organization code repository
3. official documentation
4. official dataset or benchmark page
5. maintained project page
6. secondary summary page

If a link cannot be verified, do not add it.

## Description Style

Descriptions should be:

- factual;
- concise;
- neutral;
- specific about the method, task, or contribution.

Avoid unsupported claims such as "best", "must-read", "revolutionary", or "state-of-the-art" unless the wording is tied to a specific benchmark statement from the cited source.

## Maintenance

The list should be reviewed periodically for:

- broken links;
- duplicated entries;
- outdated descriptions;
- archived repositories;
- moved paper or project pages;
- inconsistent English and Chinese content.
