# 资源模板

## 论文

```md
- **论文标题** - 作者，会议/期刊 年份。一句话客观说明方法、任务或贡献。[[论文](link)] [[代码](link)]
```

## 代码库

```md
- [仓库名称](link) - 与仓库主题相关的工具或实现，说明范围和主要使用场景。
```

## 教程

```md
- [教程标题](link) - 覆盖某个概念、实现细节或应用方向的学习资源。
```

## 数据集或 Benchmark

```md
- [数据集或 Benchmark 名称](link) - 面向某个任务的评估资源，可说明领域、模态和访问方式。
```

## 应用

```md
- **应用论文或项目** - 作者，会议/期刊 年份。核心主题在具体领域或任务中的应用。[[论文](link)] [[代码](link)]
```

## 描述规则

- 描述必须客观。
- 说明资源为什么属于本列表。
- 避免宣传语言。
- 优先使用原始来源。
- 不收录未核验链接。
