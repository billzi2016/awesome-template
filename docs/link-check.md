# Link Check

Use this checklist before adding or updating resources.

## Required Checks

- The link opens successfully.
- The link points to the intended resource.
- Title, authors, venue, year, or repository identity match the linked page.
- Code links point to the official author or organization repository when possible.
- Dataset and model links point to official pages rather than mirrors.
- The resource does not require copying protected content into this repository.

## Preferred Sources

1. arXiv, DOI, publisher, or official paper page
2. official GitHub or organization repository
3. official project page
4. official documentation
5. official dataset or benchmark page

## Risk Signals

- Download-only pages with unclear ownership
- unofficial dataset mirrors
- model weights without license information
- copied PDFs or course material
- broken redirects
- pages unrelated to the resource title

When uncertain, do not include the resource until the source is verified.
