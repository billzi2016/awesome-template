# 链接检查

新增或更新资源前，使用本清单检查。

## 必查项

- 链接可以正常打开。
- 链接指向目标资源。
- 标题、作者、会议/期刊、年份或仓库身份与页面一致。
- 代码链接尽量指向作者或机构官方仓库。
- 数据集和模型链接指向官方页面，而不是镜像站。
- 资源不需要把受保护内容复制到本仓库。

## 优先来源

1. arXiv、DOI、出版社页面或官方论文页
2. 官方 GitHub 或机构仓库
3. 官方项目页面
4. 官方文档
5. 官方数据集或 benchmark 页面

## 风险信号

- 只有下载入口且所有权不清楚的页面
- 非官方数据集镜像
- 没有许可证说明的模型权重
- 复制的 PDF 或课程材料
- 异常跳转
- 页面内容和资源标题不匹配

不确定时，不要收录，先核验来源。
